Descripción del Proyecto

Este proyecto consiste en una aplicación web desarrollada en **Python con Flask** que demuestra los conceptos fundamentales del **Despliegue Automático (CI/CD)** utilizando **Azure App Service** y **GitHub Actions**.

Objetivos del Proyecto

1.  Crear una aplicación web sencilla con mensaje de bienvenida
2.  Utilizar Python con Flask como framework
3.  Desplegar en Azure App Service
4.  Configurar despliegue automático mediante Push a GitHub
5.  Demostrar el proceso de CI/CD con GitHub Actions
